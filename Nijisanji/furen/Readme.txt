//=========================================================
// OBSでの使い方
//=========================================================
①好きなシーンで、ブラウザを追加してください。
②ブラウザソースの設定から、まず「ローカルファイル」にチェックをいれ、srcフォルダの中にある[index.html]を指定してください
③サイズは基本800×600
④カスタムCSSはそのまま何もいじらないでください
⑤OKを押せば、背景が抜けているデジタル時計が表示します。
